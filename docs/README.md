# Documentation Specifications

The core specifications and context for the Agentic Workflow Compiler are split into the following modular documents:

1. [Overall Context](./01-overall-context.md) - Vision, problem statement, core thesis, and long-term roadmap.
2. [Architecture and Models](./02-architecture-and-models.md) - Two-stage architecture, conceptual agent roles, execution models, discovery/workflow schemas, sandbox and caching.
3. [Hackathon Scope and Demo](./03-hackathon-scope-and-demo.md) - MVP scope, sponsor strategy, and the demo script.
4. [User Interface](./04-user-interface.md) - Stage 1 discovery UI, output families, and the (separate) workspace tabs.
5. [Todos and Goals](./05-todos-and-goals.md) - SMART goals, two-stage build sequence, verifiable acceptance criteria, and prioritised checklists.
6. [Two-Stage Pipeline](./06-two-stage-pipeline.md) - **Canonical MVP spec**: Paste URLs -> discover (browser agent) -> choose Document/Collection -> build a reusable workflow.
